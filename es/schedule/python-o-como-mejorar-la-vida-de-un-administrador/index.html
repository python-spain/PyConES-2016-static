Acerca de

Se ilustrará como Python puede convertirse en la herramienta ideal para la administración de sistemas sobre entornos *nix. Compararemos el código con su equivalente en bash comentando sus ventajas y posibles inconvenientes .

No se requiere experiencia previa en Python pero si es conveniente un conocimiento básico de la shell *nix y la programación de scripts.

El código usado en la charla está alojado en el siguiente repositorio de GitHub.
