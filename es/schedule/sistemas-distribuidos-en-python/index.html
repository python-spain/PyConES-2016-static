Sistemas distribuidos en Python

Guillem Borrell Nogueras

  • 7 de Octubre de 2016, 18:00 - 20:30
  • Aulario III - 5 - (0.090)
  • Idioma: es

Durante el taller se tratarán conceptos básicos y no tan básicos para poder desarrollar aplicaciones paralelas de alto rendimiento a nivel general, como el uso del sistema de ficheros, las colas de mensajes, proxies, brokers, extensiones escritas en otros lenguajes de programación y la compresión en memoria entre otros.

Si se quiere seguir la sesión será necesario instalar uwsgi, pyzmq, Cython, numpy, matplotlib y Julia.

No son necesarios conocimientos previos específicos, pero sí un nivel relativamente alto de programación en Python para poder seguir los algoritmos que se van utilizando en cada paso.
